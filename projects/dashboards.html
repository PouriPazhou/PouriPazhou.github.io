Power BI Dashboards

Skills: KPI design • Data modeling • Basic DAX • Interactive filtering

A set of dashboards demonstrating KPI cards, interactive slicers, and visual storytelling. Each dashboard focuses on a clear question and highlights key insights.

1) IMDB Movies Dashboard

  • Question: How do ratings relate to duration and share of movies?
  • Features: KPI cards, pie + bar charts, slicers

2) Average Age Leaving Parental Home (Europe, 2022)

  • Question: Which countries have the highest average leaving age?
  • Features: ranked bars, conditional formatting style

3) CO₂ Emissions by Model Year (2020–2024)

  • Question: How do average emissions change across model years?
  • Features: trend, KPIs, slicers

4) Global Emissions Share (Interactive View)

  • Question: Which countries contribute the most to emissions share?
  • Features: treemap, KPIs, interactive country selection
